# Store screenshots

Chrome Web Store requires at least **one** screenshot.

## Capture steps

1. `cd extension && npm run build`
2. Load `extension/dist` in Chrome
3. Open monitor → Begin Unshrimping → calibrate
4. Resize window to ~**1280×800** (or capture at 2× and downscale)
5. Save PNGs here with clear names:

| Filename | Content |
|----------|---------|
| `01-monitor-good.png` | Upright posture, green Shrimp Index |
| `02-monitor-slouch.png` | Slouch / alert / orange or red index |
| `03-setup.png` | Quick setup or calibration (optional) |

## Sizes

- **Required:** 1280×800 or 640×400
- **Promo tile (optional):** 440×280
- **Marquee (optional):** 1400×560

Upload in the Chrome Web Store dashboard under **Store listing → Screenshots**.
